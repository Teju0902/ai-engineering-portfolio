# Challenges

## Olivia Woodruff Portfolio

In this project, the goal was to use CSS visual rules to improve the appearance of a photography portfolio website.

### Challenge 1: Style the Header

* Review the HTML structure in `index.html`.
* Select the `.header` element.
* Set the background color to `CornflowerBlue`.

**Expected CSS:**

```css
.header {
  background-color: CornflowerBlue;
}
```

---

### Challenge 2: Center Header Content

* Modify the `.header` selector.
* Center all text within the header section using the `text-align` property.

**Expected CSS:**

```css
.header {
  text-align: center;
}
```

---

### Challenge 3: Increase About Me Text Size

* Select the `.about-me` element.
* Set the font size to `20px`.

**Expected CSS:**

```css
.about-me {
  font-size: 20px;
}
```

---

### Challenge 4: Add Transparency

* Modify the `.about-me` selector.
* Make the text 50% transparent using the `opacity` property.

**Expected CSS:**

```css
.about-me {
  opacity: 0.5;
}
```

---

### Challenge 5: Make Project Titles Bold

* Select all elements with the `.title` class.
* Make the text bold.

**Expected CSS:**

```css
.title {
  font-weight: bold;
}
```

---

### Challenge 6: Change Main Heading Color

* Select all `h1` elements.
* Set the text color to `Azure`.

**Expected CSS:**

```css
h1 {
  color: Azure;
}
```

---

### Challenge 7: Update Website Font

* Select the `body` element.
* Change the font family to `Georgia`.

**Expected CSS:**

```css
body {
  font-family: Georgia;
}
```

---

### Challenge 8: Add a Background Image

* Modify the `body` selector.
* Add the following image as the page background:

```text
https://content.codecademy.com/courses/learn-css-selectors-visual-rules/hypnotize_bg.png
```

**Expected CSS:**

```css
body {
  background-image: url("https://content.codecademy.com/courses/learn-css-selectors-visual-rules/hypnotize_bg.png");
}
```

---

### Challenge 9: Personalize the Design

* Continue experimenting with CSS visual rules.
* Customize colors, fonts, spacing, and other design elements to create a unique portfolio appearance.
