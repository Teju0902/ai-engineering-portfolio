# Olivia Woodruff Portfolio

A beginner-friendly CSS project that focuses on applying visual styling rules to enhance the appearance of a photography portfolio website.

## Project Overview

This project demonstrates the use of CSS visual rules to improve the design and layout of a portfolio webpage. The goal is to practice styling HTML elements using selectors, typography, colors, transparency, alignment, and background images.

## Features Implemented

### 1. Styled the Header Section

Applied a background color and centered the content:

```css
.header {
  background-color: CornflowerBlue;
  text-align: center;
}
```

### 2. Enhanced the About Me Section

Increased the text size and added transparency:

```css
.about-me {
  font-size: 20px;
  opacity: 0.5;
}
```

### 3. Made Project Titles Bold

Styled all project title elements:

```css
.title {
  font-weight: bold;
}
```

### 4. Updated Main Heading Color

Changed the main heading color to Azure:

```css
h1 {
  color: Azure;
}
```

### 5. Customized Page Typography

Applied the Georgia font family to the entire page:

```css
body {
  font-family: Georgia;
}
```

### 6. Added a Background Image

Enhanced the page appearance with a patterned background:

```css
body {
  background-image: url("https://content.codecademy.com/courses/learn-css-selectors-visual-rules/hypnotize_bg.png");
}
```
