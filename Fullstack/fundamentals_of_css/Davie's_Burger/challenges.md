# Davie's Burgers - CSS Box Model Lab

## Overview

This project focuses on applying CSS Box Model concepts to improve the layout and appearance of a fictional restaurant website called **Davie's Burgers**.

The HTML structure is already provided, and most CSS selectors are in place. The goal is to complete the missing box-model properties by adjusting element dimensions, spacing, borders, and overflow behavior.

## Learning Objectives

By completing this lab, you will practice:

* Setting element width and height
* Using margin for spacing and centering
* Applying padding to create internal spacing
* Adding borders to elements
* Managing content overflow
* Understanding how the CSS Box Model affects page layout

## Key CSS Concepts Practiced

### Margin

Creates space outside an element's border.

```css
margin: 40px auto;
```

### Padding

Creates space inside an element between content and border.

```css
padding: 20px;
```

### Border

Defines the visible edge of an element.

```css
border: 1px solid blue;
```

### Overflow

Controls how content behaves when it exceeds container dimensions.

```css
overflow: scroll;
```
