# Daily Buzz

## Overview

Daily Buzz is a command-line project that focuses on practicing Linux filesystem navigation and management. The project simulates organizing content for a national newspaper by creating directories and files using terminal commands.

## Objectives

* Navigate through nested directories.
* Create directories using mkdir.
* Create files using touch.
* Practice working with relative paths.
* Verify filesystem changes using ls and pwd.

## Directory Structure Created

daily-buzz/
├── national/
│   └── politics/
│       └── elections/
│           ├── candidates.txt
│           └── issues.txt
└── business/
└── startups/
└── disruptors/
├── tech.txt
├── design.txt
└── education.txt

## Commands Used

* pwd
* ls
* cd
* mkdir
* touch

## Key Learning Outcomes

* Understanding directory navigation.
* Creating nested directories.
* Managing files from different locations.
* Using relative paths effectively.
* Working efficiently within a terminal environment.
