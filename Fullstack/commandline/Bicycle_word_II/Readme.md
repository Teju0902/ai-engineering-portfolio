# Bicycle World II

## Overview

Bicycle World II is a command-line project focused on practicing basic filesystem navigation and manipulation using Linux terminal commands.

The project involves:

* Navigating directories
* Listing files and folders
* Creating files
* Creating directories
* Using relative paths

## Directory Structure

bicycle-world-ii/
├── brands.txt
├── freight/
│   ├── messenger/
│   └── porteur/
├── mountain/
│   ├── downhill/
│   │   ├── heavyweight/
│   │   └── lightweight/
│   └── hardtail/
└── racing/
├── road/
└── track/

## Commands Used

* pwd
* ls
* cd
* mkdir
* touch

## Tasks Completed

1. Displayed the current working directory.
2. Listed files and directories.
3. Navigated through the filesystem.
4. Created new files:

   * dirt.txt
   * mud.txt
   * tricks.txt
5. Created new directories:

   * safety/
   * bmx/
6. Verified changes using the ls command.

## Learning Outcomes

Through this project, I learned how to:

* Navigate a filesystem using the command line.
* Create and organize directories.
* Create files from the terminal.
* Use relative paths efficiently.
* Understand Linux filesystem hierarchy.
