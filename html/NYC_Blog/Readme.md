# New York City Blog

## Project Overview

This project is a semantic HTML webpage about New York City. It was created as part of a Semantic HTML challenge to practice using meaningful HTML elements that improve accessibility, readability, and webpage structure.

The webpage includes:

* A navigation bar with links to different sections
* A blog article about New York City
* An image of the Statue of Liberty with a caption
* An aside section featuring popular New York City foods
* Media content including video, image, and audio
* A footer with author and contact information

## Technologies Used

* HTML5
* Semantic HTML Elements

## Semantic Elements Used

* `<nav>`
* `<header>`
* `<main>`
* `<section>`
* `<article>`
* `<figure>`
* `<figcaption>`
* `<aside>`
* `<video>`
* `<audio>`
* `<embed>`
* `<footer>`

## Learning Objectives

This project demonstrates:

* Proper use of semantic HTML elements
* Page organization and structure
* Internal navigation using anchor links
* Embedding multimedia content
