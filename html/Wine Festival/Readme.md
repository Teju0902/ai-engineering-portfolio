# Wine Festival Schedule

## Project Overview

This project is a simple HTML table that displays the schedule for the annual Aguillar Family Wine Festival. The goal of this lab activity is to practice creating and organizing tables in HTML using table-related elements such as `<table>`, `<thead>`, `<tbody>`, `<tr>`, `<th>`, and `<td>`.

## Objectives

* Create a structured HTML table.
* Use table headers and table body sections.
* Organize event information into rows and columns.
* Apply the `colspan` attribute to merge table header cells.
* Use class attributes for styling and layout purposes.

## Technologies Used

* HTML5

## Table Structure

### Table Header

The table header contains:

* A title row spanning two columns:

  * **Wine Festival Schedule**
* A column heading row:

  * **Time**
  * **Event**

### Table Body

The table body contains five event entries:

| Time     | Event                  |
| -------- | ---------------------- |
| 12:00 PM | Welcome Reception      |
| 01:00 PM | Storytelling & Tasting |
| 02:00 PM | Wine Luncheon          |
| 03:00 PM | Aguillar Family Wines  |
| 04:00 PM | Wine & Cheese Tasting  |
