# Everyday with Isa - Fashion Blog

## Project Overview

This project is a beginner-friendly HTML website created for a fictional fashion blogger, **Isa (Isabelle Rodriguez)**. The blog provides readers with fashion tips and insights about **New York Fashion Week (NYFW)** while demonstrating the use of fundamental HTML elements and page structure.

## Objectives

The purpose of this project is to practice:

* Building a complete HTML document structure
* Creating headings and paragraphs
* Adding images to a webpage
* Creating hyperlinks
* Building unordered lists
* Organizing content using semantic HTML elements

## Technologies Used

* HTML5

## Features

### Blog Header

* Website title: **Everyday with Isa**
* Main article heading: **An Insider's Guide to NYFW**
* Author information and profile image

### Blog Content

Three sections covering:

1. Introduction to New York Fashion Week
2. Getting tickets and planning schedules
3. Dressing appropriately for fashion shows

### Images

The blog includes:

* Author profile picture
* Three fashion-related images placed above each blog section

### External Link

* The term **NYFW** links to the New York Fashion Week Wikipedia page.
* The link opens in a new browser tab using the `target="_blank"` attribute.

### Related Content

An unordered list of related fashion blog topics:

* How To Style Boyfriend Jeans
* When Print Is Too Much
* The Overalls Trend
* Fall's It Color: Blush

## HTML Elements Practiced

* `<!DOCTYPE html>`
* `<html>`
* `<head>`
* `<title>`
* `<body>`
* `<h1>` to `<h4>`
* `<p>`
* `<img>`
* `<a>`
* `<ul>`
* `<li>`

## Key Concepts Learned

* Creating the basic structure of a web page
* Adding and organizing textual content
* Embedding images with the `src` attribute
* Creating hyperlinks with the `<a>` tag
* Opening links in a new tab using `target="_blank"`
* Building lists for related content
* Improving webpage readability with headings

## Learning Outcomes

By completing this project, I gained hands-on experience with fundamental HTML concepts and learned how to structure a simple blog page using headings, paragraphs, images, links, and lists.
