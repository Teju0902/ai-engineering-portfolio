# Fashion Blog

## Overview

A beginner HTML project focused on creating a simple fashion blog webpage.

## Concepts Covered

* HTML Document Structure
* Headings
* Paragraphs
* Images
* Hyperlinks
* Lists
* Semantic HTML
* Div Elements
* IDs and Anchor Links

## Features

* Blog article layout
* Author profile section
* External links
* Related content list
* Contact section
* Internal page navigation

## Learning Outcomes

Through this project, I practiced:

* Structuring a webpage using HTML
* Organizing content with semantic elements
* Working with images and hyperlinks
* Creating lists and navigation links
* Building a complete static webpage
